# Multi-Chain USDT Sender

Send USDT instantly across TRC20, ERC20, and BEP20 networks through a secure, non-custodial wallet-connect interface (MetaMask, TronLink).

## Features

- **Multi-chain support** — TRC20 (Tron), ERC20 (Ethereum), BEP20 (BNB Smart Chain)
- **Non-custodial** — users connect their own wallet; funds never pass through a third party
- **Wallet compatibility** — MetaMask (EVM chains), TronLink (Tron)
- **Fast & low-fee transfers** — optimized network selection for speed and cost

## Supported Networks

| Network | Standard | Wallet |
|---|---|---|
| Tron | TRC20 | TronLink |
| Ethereum | ERC20 | MetaMask |
| BNB Smart Chain | BEP20 | MetaMask |

## How It Works

1. Connect your wallet (MetaMask or TronLink)
2. Select the network matching your recipient's address
3. Enter recipient address and amount
4. Confirm and send

## Important

Sender and receiver must use the **same network**. Sending to a mismatched network address results in permanent loss of funds. Always verify the network before confirming a transaction.

## FAQ

**Is this custodial?**
No. This tool connects directly to your wallet — you retain full control of your funds at all times.

**What's the difference between TRC20 and ERC20?**
Both are USDT token standards on different blockchains. TRC20 (Tron) typically has lower fees and faster confirmation than ERC20 (Ethereum).

**What are the fees?**
Network gas fees apply and vary by chain. Tron (TRC20) generally has the lowest fees.
